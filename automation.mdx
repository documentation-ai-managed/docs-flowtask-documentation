## Setting Up Automations

<Steps>
  <Step title="Access Automation" icon="settings">Navigate to Project Settings > Automation tab</Step>
  <Step title="Choose Template" icon="template">Select from pre-built templates or create custom</Step>
  <Step title="Configure Trigger" icon="play">Set up what starts the automation</Step>
  <Step title="Define Actions" icon="cog">Specify what the automation should do</Step>
  <Step title="Test and Save" icon="check">Test the automation and save it</Step>
</Steps>

<Callout kind="tip">Start with simple automations and gradually add complexity as you become familiar with the system.</Callout>

## Advanced Automation Features

<CodeGroup tabs="JavaScript,Python">
  ````javascript
// Create a custom automation
const automation = await client.automations.create({
  name: 'Task Completion Follow-up',
  trigger: {
    type: 'task.completed',
    projectId: '123'
  },
  conditions: [
    { field: 'priority', operator: 'equals', value: 'high' }
  ],
  actions: [
    { type: 'send_email', to: 'manager@company.com', template: 'task_complete' }
  ]
});
  ````
  ````python
# Advanced automation with conditions
automation = client.automations.create(
    name='Task Completion Follow-up',
    trigger={
        'type': 'task.completed',
        'project_id': '123'
    },
    conditions=[
        {'field': 'priority', 'operator': 'equals', 'value': 'high'}
    ],
    actions=[
        {'type': 'send_email', 'to': 'manager@company.com', 'template': 'task_complete'}
    ]
)
  ````
</CodeGroup>

## Troubleshooting Automations

<ExpandableGroup>
  <Expandable title="Automation not triggering" default-open="false">
    Check trigger conditions and ensure all required fields are present. Verify the automation is enabled.
  </Expandable>
  <Expandable title="Actions failing" default-open="false">
    Review action configurations and check for permission issues with connected services.
  </Expandable>
  <Expandable title="Performance issues" default-open="false">
    Too many automations can slow down your workspace. Consider consolidating or disabling unused ones.
  </Expandable>
</ExpandableGroup>
