## Creating Projects

FlowTask makes project creation effortless with AI-powered assistance that helps you structure your ideas into actionable projects.

The more detailed your initial description, the better FlowTask can suggest an optimal project structure.

## Project Structure

FlowTask organizes projects hierarchically with tasks, subtasks, and milestones.

## Collaboration Features

<CodeGroup show-lines="true" tabs={[]}>
  ```javascript
  // Create a project with team members
  const project = await client.projects.create({
  title: 'Q4 Marketing Campaign',
  description: 'Launch new product marketing',
  members: ['user1@company.com', 'user2@company.com'],
  dueDate: '2024-12-31'
  });
  ```

  ```python
  # Create collaborative project
  project = client.projects.create(
    title='Q4 Marketing Campaign',
    description='Launch new product marketing',
    members=['user1@company.com', 'user2@company.com'],
    due_date='2024-12-31'
  )
  ```
</CodeGroup>

## API Integration

Integrate project data with external systems:

<Tabs>
  <Tab title="Webhooks" icon="webhook">
    Receive real-time updates when projects change.

    ```json
    {
      "event": "project.updated",
      "data": {
        "projectId": "123",
        "changes": ["status", "dueDate"]
      }
    }
    ```
  </Tab>

  <Tab title="REST API" icon="code">
    Full programmatic access to project operations.

    ```bash
    curl -X GET "https://api.flowtask.com/v1/projects/123" \
      -H "Authorization: Bearer YOUR_API_KEY"
    ```
  </Tab>
